```

[1]        [2] [3]  [4]  [5]    [6]             [7]
```

1. 文件的类型和权限  
第一个字符表示文件的类型，可以是如下字符：  
    ```
    d :表示目录
    - :表示文件
    l :表示链接文件
    b :表示随机读写设备，块级设备
    c :表示串口设备，字符设备
    ```
