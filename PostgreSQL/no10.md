## PostgreSQL学习笔记 索引

### 1. 索引的分类

|  索引名称   |             使用场景             |
| :---------: | :------------------------------: |
| B-tree 索引 | 适合处理那些能够按顺序存储的数据 |
|  Hash 索引  |      只能处理简单的等于比较      |
|  Gist 索引  |           一种索引架构           |
|  GIN 索引   |   反转索引，处理包含多个值的键   |

### 2. 索引的优点和缺点

1. 优点：
   + 提高数据的查询速度
   + 加速表与表之间的链接
2. 缺点：
   + 创建和为何索引需要耗费时间
   + 需要占用磁盘空间
